# 🌐 Portfolio Website

A personal portfolio website showcasing my projects, skills, and experience as a Frontend Developer.

---

## 🚀 Project Overview

This project was developed in multiple stages to demonstrate flexibility across different technologies:

1. **HTML/CSS Version** – Initial static portfolio design  
2. **WordPress Version** – Converted into a dynamic CMS-based website  
3. **Deployed Version** – Optimized static version deployed on Netlify  

---

## 🛠 Tech Stack

### 💻 Frontend
- HTML5  
- CSS3  
- JavaScript  

### ⚙️ CMS / Backend
- WordPress (Custom Theme Development)  

### 🌐 Deployment
- Netlify
